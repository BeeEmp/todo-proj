Hi, this is a personal learning project designed to practice DOM manipulation, LocalStorage, and CSS styling.


About:

This is a functional Todo List application built with vanilla HTML, CSS, and JavaScript. It allows users to add tasks, delete them, and persist data using the browser's LocalStorage so tasks aren't lost on refresh.

Inspiration & Source:

This project was originally based on the tutorial "Build a Todo List App with HTML, CSS & JavaScript".

Personalizations & Learning Outcomes

While the core logic follows the tutorial, I have customized the design and structure to improve the UI/UX and practice new concepts:



Technologies Used

HTML5: Semantic markup.

CSS3: Custom properties (variables), Flexbox, and responsive styling.

JavaScript: DOM manipulation and LocalStorage data persistence.

FontAwesome: For UI icons (edit/delete).

Google Fonts: Used 'Montserrat' and 'JetBrains Mono'.


How to Run?

Simply open the index.html file in any modern web browser. No installation or server required
